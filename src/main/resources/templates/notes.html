Chorus Digital Notes

These are notes about maintaining this app.

The application is a Spring Boot application (Java). It is encapsulated in a single executable jar file which is run as a service on a Linux server. The application has an embedded Tomcat web server and uses an H2 database for persistence.

The app is generally developed in JetBrains' Idea, although this is not required. It is backed up in a git repository. The app uses Gradle for a build system.

Major dependencies are:

  • spring-boot-starter-web (web controllers)
  • spring-boot-starter-jpa (persistence)
  • spring-boot-starter-thymeleaf (web page formatting)
  • spring-boot-starter-actuator (web metrics)
  • spring-boot-starter-security
  • h2 (database)
  • Other auxiliary components
